build CMVS-PMVS¶

https://github.com/pmoulon/CMVS-PMVS/issues/1

git clone https://github.com/pmoulon/CMVS-PMVS.git
mkdir CMVS-PMVS_Build
cd CMVS-PMVS_Build
cmake . ../CMVS-PMVS/program
make
sudo make install
