Install UniversalMediaServer¶

Web References¶

  • GitHub

https://github.com/UniversalMediaServer/UniversalMediaServer

Install¶

su -
apt-get install mediainfo openjdk-7-jre dcraw vlc

Download latest linux version in home directory

  • Latest linux Releases

http://sourceforge.net/projects/unimediaserver/files/Official%20Releases/Linux/

cd ~
tar xzvf UMS-5.2.2-Java7.tgz

The program is in directory ums-5.2.2

Run UniversalMediaServer¶

cd ums-5.2.2
./UMS.sh

Ums est accessible a http://localhost:9001
