................

.. image:: /images/bliss.png

Web References
..............

http://www.blisshq.com/

http://www.blisshq.com/download-thank-you-linux.html

install bliss
.............

http://www.blisshq.com/install-linux.html

.. code:: tcsh

  java -jar bliss-install-20150825.jar
  bin/bliss.sh

bliss will be running in http://localhost:3220/

re-install bliss
................

.. code:: tcsh

  ~/bliss/Uninstaller
  java -jar uninstaller.jar
  rm -r .java/
  java -jar bliss-install-20150825.jar
